# 观察日

> 带观察日是你个人发展的需要，同时也是成就他人的开始。每个观察日都可能成为你的左右手，也都有可能是你未来的经理。对待观察日不能不重视但也不能太在乎：你是机会的给予者，不必处心积虑去取悦对方，也不要用自己个人的喜好来决定其去留。

## 五、留存关键

让观察日学到东西；看到系统的成功、赚钱的可能、工作的乐趣；看到、听到、感觉到发展（榜样、拼劲、创业精神），这些才是留存的关键。
</blockquote>
