# 榜样

组长阶段是你踏入管理之路的开端，从此阶段起，你最重要的是做好的榜样，同时把你学到的经验传承给你的队员、激励你的队员。最直接有效的方法便是以身作则，用你的热忱、认真的态度、良好的品格来影响他人。

同创主悦的价值观是正面积极的，作为同创人，人要正。榜样的力量无穷，生命可以影响生命，因此作为组长更要做好“正”的榜样。身正取决于心正，心正取决于内心的底气，底气来源于自身的“功”与“德”，无论是工作中还是生活中，都要严格要求自己，使队员信服。

**作为榜样的“影响”要有成果，要靠“修业、进德”。**

* 修业--做好工作榜样：扩大你的战果、提高你的成果。事实胜于雄辩，工作中自己要以切实行动影响队员，做出实际成果为队员树立工作标准。队员心中都有一杆工作标准的秤，秤的一边是你讲的道理，另一边是你的实际成果，只有当成果的重量等于甚至大于道理的重量之时，队员才会产生信服，你才开始具有工作上的榜样性。
* 进德--做好生活榜样：提升你的品格、认知、格局、高度。生活与工作密不可分、互相影响。良好习惯成就美好人生，习惯形成于日常生活。作为组长，日常生活应做到自律，热衷于提升个人综合素质，成为团队的道德标杆。


## 榜样的六大准则：

1. 成功品格

尊重、利他、共赢

2. 良好的态度

谦卑、顺服、好学、愿意改变

3. 责任感

·用持续的钟声营造公司氛围；

·培训好拍档；

·很好地协调解决队员的问题；

·能够努力完成早晚黄金时间所订的目标；

·很好地完成公司所安排的特别任务；

·爱护好公司的货物；

·困难时刻仍然具备对生意负责任的态度。

·新产品上市要主动带动、了解。

4. 学习与教育的能力

用未来身份做现在事。作为一个老师，懂得有效地教育队员的五步、八点、平均法；

为完善制度提供建议；了解并推广企业文化及未来的发展方向等。

5. 稳定的赚钱能力

组长=完成任务：水准不低于打钟的一半。

6. 独立作业的能力

有能力自我鼓励，并懂得安排地区作业；新产品的推广；主动要求自己好上加好、高标准。
