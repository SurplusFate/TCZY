# 💪 鼓励
> 同创主悦是一条通往美好生活的道路，这条路上充满鲜花和掌声，同时也会有挫折、问题、负面情绪。作为已经走过这条路的前辈，更知道如何激励正在经历失败、低谷的队员；以切身经验体会告知队员遇到负面、处于瓶颈期时应该具有的思想、应该做出的行动，让队员明白：遇到困难要思考3个问题：你可以做什么——你能做的；要做什么——你想做的；该做什么——属于你责任范围内的。

---
