## 📅 如何带新人

<div id="section-how" style="background:rgba(102,126,234,0.06);border:1px solid rgba(102,126,234,0.15);padding:20px;border-radius:16px;margin:20px 0;">

<p style="margin:0 0 15px 0;font-size:15px;color:#333;line-height:1.8;">带新人不应拘泥于固定时间、地点、形式，应充分利用早晚黄金时间段、作业中的任何机会，使其尽快熟悉工作内容、融入工作环境。</p>

</div>

<div style="display:grid;grid-template-columns:repeat(auto-fit,minmax(300px,1fr));gap:15px;margin:20px 0;">

<div style="background:rgba(102,126,234,0.08);border:1px solid rgba(102,126,234,0.15);padding:20px;border-radius:12px;">
<p style="margin:0 0 12px 0;font-size:18px;font-weight:bold;color:#667eea;">🌅 早</p>
<div style="display:flex;flex-direction:column;gap:8px;">
<span style="background:rgba(102,126,234,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">比他早到公司准备好今天的安排并迎接他。</span>
<span style="background:rgba(102,126,234,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">请他协助一些工作。</span>
<span style="background:rgba(102,126,234,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">共同练习五步。</span>
<span style="background:rgba(102,126,234,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">开会时站在他旁边、解释开会内容、介绍台上的人。</span>
<span style="background:rgba(102,126,234,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">自己因其他事物无法关注新人时，请其他优秀的同事替代。</span>
</div>
</div>

<div style="background:rgba(34,197,94,0.08);border:1px solid rgba(34,197,94,0.15);padding:20px;border-radius:12px;">
<p style="margin:0 0 12px 0;font-size:18px;font-weight:bold;color:#22c55e;">💼 作业中</p>
<div style="display:flex;flex-direction:column;gap:8px;">
<span style="background:rgba(34,197,94,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;"><strong>乐趣</strong>：作业不是苦差事，轻松作业、乐趣作业的同时可得到成长。</span>
<span style="background:rgba(34,197,94,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;"><strong>正面积极的思维</strong>：面对、解决负面状况。</span>
<span style="background:rgba(34,197,94,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;"><strong>额外里程</strong>：完成目标的决心。</span>
</div>
</div>

<div style="background:rgba(245,158,11,0.08);border:1px solid rgba(245,158,11,0.15);padding:20px;border-radius:12px;">
<p style="margin:0 0 12px 0;font-size:18px;font-weight:bold;color:#f59e0b;">🌙 晚</p>
<div style="display:flex;flex-direction:column;gap:8px;">
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">推广打钟的新人（如：推广此新人在作业过程中感到得意的事情）；其他人打钟时向新人推广这个打钟的同事。</span>
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">针对今日作业的可加强之处予以练习及教育。</span>
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">教育企业文化、推广会议的照片或资料。</span>
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">与其他同事间的相互推广。</span>
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">推动他追求发展的渴望。</span>
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">在他最兴奋的时候让他离开公司，不要留他到太晚。</span>
<span style="background:rgba(245,158,11,0.08);padding:8px 12px;border-radius:8px;font-size:14px;color:#333;">向经理汇报他今天的状况。</span>
