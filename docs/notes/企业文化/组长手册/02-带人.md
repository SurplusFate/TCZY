# 🤝 带人
> 全新的工作环境、工作内容总是伴随着不安、焦虑等负面情绪的产生。作为组长，能否及时察觉新人的情绪波动，并采取行之有效的改善措施是带好新人的关键。所谓带人，不仅是带领，还要会交朋友；所谓照顾，不是给予特殊关照，而是让他感受到你的时刻关注。

## 📅 如何带新人

<div id="section-how" class="tc-container tc-container-blue">

<p class="tc-body" style="margin:0 0 15px 0;font-size:15px;line-height:1.8">带新人不应拘泥于固定时间、地点、形式，应充分利用早晚黄金时间段、作业中的任何机会，使其尽快熟悉工作内容、融入工作环境。</p>
