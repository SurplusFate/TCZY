# 带人

全新的工作环境、工作内容总是伴随着不安、焦虑等负面情绪的产生。作为组长，能否及时察觉新人的情绪波动，并采取行之有效的改善措施是带好新人的关键。所谓带人，不仅是带领，还要会交朋友；所谓照顾，不是给予特殊关照，而是让他感受到你的时刻关注。


## 如何带新人

带新人不应拘泥于固定时间、地点、形式，应充分利用早晚黄金时间段、作业中的任何机会，使其尽快熟悉工作内容、融入工作环境。


早

· 比他早到公司准备好今天的安排并迎接他；

· 请他协助一些工作；

· 共同练习五步；

· 开会时站在他旁边、解释开会内容、介绍台上的人；

· 自己因其他事物无法关注新人时，请其他优秀的同事替代。

作业中

· 乐趣：作业不是苦差事，轻松作业、乐趣作业的同时可得到成长。

· 正面积极的思维：面对、解决负面状况。

· 额外里程：完成目标的决心。

晚

· 推广打钟的新人（如：推广此新人在作业过程中感到得意的事情）；其他人打钟时向新人推广这个打钟的同事；

· 针对今日作业的可加强之处予以练习及教育；

· 教育企业文化、推广会议的照片或资料；

· 与其他同事间的相互推广；

· 推动他追求发展的渴望；

· 在他最兴奋的时候让他离开公司，不要留他到太晚；

· 向经理汇报他今天的状况。

## 教育内容应注意五个方面：

一、清楚细节——新人不同阶段的教育

| 进公司的时间 | 教育重点 | 目的 |
| :--- | :--- | :--- |
| **一周内** | 对环境的适应 对你的认可 对企业的认同 | 建立新人对企业的初步认识，增加人员粘性。 |
| **二周内** | 经营系统：五步、八点、平均法 公司认知：公司运作用语、企业文化、企业优势、成功案例 工作重心：基础生意系统学习，而非仅仅关注成果 评估体系：公司对个体的关注与评估 | 尽快学会初步运作，建立对生意系统的信心。使其认可企业文化、明白事业的成功要从外出作业开始。重视学习与改变的重要性，从实际工作中获得经验并建立信心。适度给予紧迫感，使其重视自我管理及约束、更珍惜机会。 |
| **三周内** | 经营系统：对五步、八点、平均法更深层次的理解与运用；“冲动”的运用；伙伴系统--对抗负面、激励与带动 规划升组长的目标：为什么要升组长、组长需具备的条件和责任、设定升组长的时间计划 突破的重要性及如何突破 独立性：作业、解决问题、判断力 | 提高作业能力，建立正确的生意态度。 提高积极性，了解企业晋升制度。 提高作业能力 |
| **四周内** | 经营系统：系统化与灵活相结合的工作方式；伙伴系统的深入理解管理方面的初步认识：带队组长需具备的条件与责任；榜样的重要性；如何鼓励与激发同事将来身份做现在事：观察学习组长的日常工作内容----主动参与气氛营造；尝试向比自己更新的同事推广公司、分享经验 | 提高参与感与积极性 |

二、因材施教——根据不同新人的情况做一些相对应的教育来进行调整

三、了解你教育的效果——认真做好跟踪教育

四、别忘记你也曾经是个新手——多点耐心和细心

五、让他知道你对他的器重——表现出你对他的期望
