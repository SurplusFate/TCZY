# 公司为什么从外出作业开始——个人成长（上）

## 一、 核心答案：这是一种考验与筛选
*   **公司的选人第一标准**：不看重学历、经验、背景或能力。首要且唯一的问题是：**“能不能接受跑外勤？”**
*   **筛选逻辑**：能坚定接受外勤工作，本身就证明了一种“优秀的点”或“能力”——愿意从最基础、最辛苦的业务做起。
*   **现实基础**：很多人选择这份工作，是因为在当前条件下，这是能获得相应报酬的“最优解”或“唯一选择”。既然为了赚钱，就需要接受相应的工作内容。

## 二、 深层原因：外勤是一场“炼心”的修行
外勤工作的本质，是**对人“心性”的修炼和考验**，目的是锻造能克服人性劣根性的人。

## 三、 对员工的现实提醒：认清自己与当下
1.  **频繁“换泳池”解决不了根本问题**：
    *   就像不会游泳的人，换再多泳池还是不会。如果自身能力、心性不过关，换任何工作都难有起色。
    *   真正的成长，是在一个地方“学会游泳”，即沉下心来克服困难，提升自己。

2.  **“难”是正常的，也是价值的体现**：
    *   有价值的事情都难。难，往往不是因为工作本身，而是因为“当下对工作的理解和态度”——即不懂、不会、不敢开始。
    *   所谓的“没经验”、“不适合”、“嘴笨”，往往是“不敢尝试”、“懒惰”、“放不下面子”、“不想动脑”的借口。

## 四、 总结
公司通过艰苦的**外勤工作**，作为**炼金石和培养皿**。目的是：
*   **筛选**出那些心性坚定、能吃苦、愿意从基层做起的人。
*   **锻造**员工克服人性弱点（贪、惰、嗔、玩、痴），培养抗压、自律、着眼长远的稳健心态。
*   **践行“以人为本”**：公司看重对人的培养。目标是训练出能担当重任的“将才”，而“一将功成万骨枯，千金易得一将难求”，严格的考验是必要的。

**最终导向**：这不是一份轻松的工作，而是一场持续的“心性”修炼。能通过考验并坚持下来的人，获得的不仅是收入，更是内在的成长与蜕变。
