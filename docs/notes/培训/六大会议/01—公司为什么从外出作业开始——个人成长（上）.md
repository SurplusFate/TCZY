# 公司为什么从外出作业开始——个人成长（上）

## 一、 核心答案：这是一种考验与筛选
*   **公司的选人第一标准**：不看重学历、经验、背景或能力。首要且唯一的问题是：**“能不能接受跑外勤？”**
*   **筛选逻辑**：能坚定接受外勤工作，本身就证明了一种“优秀的点”或“能力”——愿意从最基础、最辛苦的业务做起。
*   **现实基础**：很多人选择这份工作，是因为在当前条件下，这是能获得相应报酬的“最优解”或“唯一选择”。既然为了赚钱，就需要接受相应的工作内容。

## 二、 深层原因：外勤是一场“炼心”的修行
外勤工作的本质，是**对人“心性”的修炼和考验**，目的是锻造能克服人性劣根性的人。

### 需要克服的五大人性劣根性：
1.  **贪**：欲望过重，唯利是图，不择手段。
    *   *职场表现*：为短期利益欺骗客户、违规操作（如贩卖电话卡）。
    *   *后果*：看似短期收入高，但售后风险巨大，个人信誉破产，不可持续。
    *   *公司倡导*：规范、正规作业。“拿得稳、拿得长久”远比“拿得快、拿得多”更重要。

2.  **惰**：懒惰，不努力、不思考、不主动。
    *   *表现*：工作上不愿动脑（如敷衍写话术）、不愿请教、不愿承担。
    *   *本质*：思想的懒惰延伸到行为的懒惰。总将问题归咎于外部（公司、工作），而非自身。

3.  **嗔**：易怒、抱怨、情绪不稳定、控制不住脾气。
    *   *表现*：因天气、小事动怒，将个人情绪带到工作中，甚至顶撞管理。
    *   *观点*：连自己脾气都控制不住的人，难成大事。公司不需要“有本事有脾气”更不容忍“没本事有脾气”的人。

4.  **玩**：贪玩，心不定，难以收心投入正事。
    *   *分析*：人皆贪玩，但“没玩过”才向往。当你有钱有时间时，反而可能“玩腻了”，最终仍需踏实做事。
    *   *现状*：正因当下没条件玩，才要克制玩心，专注工作。羡慕他人玩乐是人之常情，但要认清自身阶段。

5.  **痴**：不明事理，缺乏基本认知和常识，不讲道理。
    *   *举例*：工作中遇到无法沟通、不讲理的客户。
    *   *启示*：社会充满复杂性，需要提升认知，学会与不同人打交道。

## 三、 对员工的现实提醒：认清自己与当下
1.  **频繁“换泳池”解决不了根本问题**：
    *   就像不会游泳的人，换再多泳池还是不会。如果自身能力、心性不过关，换任何工作都难有起色。
    *   真正的成长，是在一个地方“学会游泳”，即沉下心来克服困难，提升自己。

2.  **“难”是正常的，也是价值的体现**：
    *   有价值的事情都难。难，往往不是因为工作本身，而是因为“当下对工作的理解和态度”——即不懂、不会、不敢开始。
    *   所谓的“没经验”、“不适合”、“嘴笨”，往往是“不敢尝试”、“懒惰”、“放不下面子”、“不想动脑”的借口。

## 四、 总结
公司通过艰苦的**外勤工作**，作为**炼金石和培养皿**。目的是：
*   **筛选**出那些心性坚定、能吃苦、愿意从基层做起的人。
*   **锻造**员工克服人性弱点（贪、惰、嗔、玩、痴），培养抗压、自律、着眼长远的稳健心态。
*   **践行“以人为本”**：公司看重对人的培养。目标是训练出能担当重任的“将才”，而“一将功成万骨枯，千金易得一将难求”，严格的考验是必要的。

**最终导向**：这不是一份轻松的工作，而是一场持续的“心性”修炼。能通过考验并坚持下来的人，获得的不仅是收入，更是内在的成长与蜕变。
