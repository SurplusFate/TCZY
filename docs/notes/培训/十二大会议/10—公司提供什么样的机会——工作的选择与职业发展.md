<div class="tc-body" style="background:rgba(102,126,234,0.06);border:1px solid rgba(102,126,234,0.15);padding:20px;border-radius:12px;flex:1;min-width:280px">
<p class="tc-blue tc-bold" style="font-size:24px;margin:0 0 12px 0">1️⃣ 行业前景</p>
<div style="background:rgba(102,126,234,0.08);padding:12px;border-radius:8px;border-left:4px solid #667eea">
<p style="font-size:14px;margin:0 0 8px 0;line-height:1.6"><strong>判断标准</strong>：</p>
<ul style="margin:0;padding-left:18px;font-size:14px;line-height:1.6">
<li>行业是否在增长？</li>
<li>市场需求是否持续？</li>
<li>是否有政策支持？</li>
</ul>
<p style="font-size:14px;margin:8px 0 0 0;line-height:1.6"><strong>避坑</strong>：选择夕阳行业，再努力也难有发展。</p>
</div>
</div>

<div class="tc-body" style="background:rgba(34,197,94,0.06);border:1px solid rgba(34,197,94,0.15);padding:20px;border-radius:12px;flex:1;min-width:280px">
<p class="tc-green tc-bold" style="font-size:24px;margin:0 0 12px 0">2️⃣ 平台机制</p>
<div style="background:rgba(34,197,94,0.08);padding:12px;border-radius:8px;border-left:4px solid #22c55e">
<p style="font-size:14px;margin:0 0 8px 0;line-height:1.6"><strong>判断标准</strong>：</p>
<ul style="margin:0;padding-left:18px;font-size:14px;line-height:1.6">
<li>晋升机制是否公平透明？</li>
<li>收入与付出是否成正比？</li>
<li>是否有学习和成长的机会？</li>
</ul>
<p style="font-size:14px;margin:8px 0 0 0;line-height:1.6"><strong>避坑</strong>：选择"天花板"低、晋升靠关系的平台。</p>
</div>
</div>

<div class="tc-body" style="background:rgba(245,158,11,0.06);border:1px solid rgba(245,158,11,0.15);padding:20px;border-radius:12px;flex:1;min-width:280px">
<p class="tc-orange tc-bold" style="font-size:24px;margin:0 0 12px 0">3️⃣ 个人匹配</p>
<div style="background:rgba(245,158,11,0.08);padding:12px;border-radius:8px;border-left:4px solid #f59e0b">
<p style="font-size:14px;margin:0 0 8px 0;line-height:1.6"><strong>判断标准</strong>：</p>
<ul style="margin:0;padding-left:18px;font-size:14px;line-height:1.6">
<li>是否符合自己的性格和优势？</li>
<li>是否能接受工作的挑战？</li>
<li>是否与自己的目标一致？</li>
</ul>
<p style="font-size:14px;margin:8px 0 0 0;line-height:1.6"><strong>避坑</strong>：选择与自己不匹配的工作，难以坚持。</p>
