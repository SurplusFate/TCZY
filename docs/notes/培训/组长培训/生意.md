# 从“产品生意”到“同创生意”

## 一、核心理念：两种生意的本质区别
*   **产品生意**：本质是卖货。
*   **同创生意**：本质是利用产品作为“工具”，去学习和构建一个**可复制、可传承、可壮大**的事业系统。

## 二、核心观点阐述
### 1. 成功如同金字塔：补齐根基，方能登顶
*   可见的“塔尖”（职位、业绩）由下面看不见的“根基”决定。
*   **根基**是日常的销售工作、拜访经验、成交流程（如5864、销售五步）以及对企业文化（愿景、使命、价值观）的深刻理解。
*   对企业文化的认同程度，决定了你职位的高度。

### 2. 从体力赚钱到经验赚钱：个人成长的跃迁
*   高层（如“军长”）不需要自己“打枪”，而是依靠统领全局的“经验”。
*   同样的地区、产品、话术，业绩不同的核心在于是否用心、是否懂得“讲故事”让产品“活”起来。

### 3. 心智成熟是内心强大的表现
*   **成熟的标志**：能够接受之前不能接受的事情。
*   在团队中要学会妥协，与不喜欢的人和事共处，允许差异的存在。这不是软弱，而是包容的格局。
*   **内心强大是练出来的**：通过主动去做不敢做的事（如陌生拜访、挑战权威）来磨练。逃避问题是懦弱，直面问题才能成长。

### 4. 事业的“附加值”比收入本身更重要
*   如同城市房产比农村自建房价值高，是因为附加了教育、医疗、人脉等资源。
*   在“同创生意”中，真正的“附加值”在于：
    *   **敬畏之心**：对事业、规则保持敬畏，而非散漫随意。
    *   **生命影响生命**：成就他人，用心教授，组建和壮大团队。
    *   **营造积极氛围**：以身作则，传递正能量，带动他人。
*   拥有这些附加值，个人价值才会升高，才能获得真正的尊重和长远发展。

## 三、对团队成员的要求
1.  **要有野心，敢于展现**：不要隐藏事业心，用行动和结果去争取。
2.  **要有公心，用心教人**：怕麻烦就带不好团队。组建团队是为了共同成就，团队的壮大是你成功的前提。
3.  **要严谨务实，戒除算计**：扎实做好每一个销售动作，不要在小事上斤斤计较（如一瓶水、一句话），心思要放在事业上。
4.  **要有担当，不谋财害命**：如果没有强烈的事业心和持续的努力，就不要带人，否则是在浪费他人的时间和生命。

## 四、总结
这是一门需要**用产品为工具，构建可传承系统**的“同创生意”。它要求我们以创业者心态，扎实根基、锻造强大内心、并创造超越业绩的附加值。唯有如此，才能获得维持奋斗一生的事业，而不仅仅是一份工作。
