## ❓ 核心问题与目标

## 🤝 一、新人入职初期状态与对策

<div style="display:flex; flex-wrap:wrap; gap:12px; margin-bottom:16px;">
  <div style="background:rgba(239,68,68,0.08); padding:16px; border-radius:12px; flex:1; min-width:280px; border:1px solid rgba(239,68,68,0.15);">
    <h4 style="color:#ef4444; margin-top:0;">😟 新人状态</h4>
    <ul style="margin:0; padding-left:20px;">
      <li>对工作完全陌生，缺乏安全感</li>
      <li>仅认识少数同事</li>
    </ul>
  </div>
  <div style="background:rgba(34,197,94,0.08); padding:16px; border-radius:12px; flex:1; min-width:280px; border:1px solid rgba(34,197,94,0.15);">
    <h4 style="color:#22c55e; margin-top:0;">✅ 应对策略</h4>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px; margin-bottom:8px;">
      <p style="margin:0;"><strong>1. 营造氛围</strong>：引导新人关注工作群内的成交喜报，借机宣传公司的成功案例和团队文化</p>
      <p style="margin:8px 0 0 0; font-size:13px; font-style:italic;">例如：做市场的时候可以时不时看下群里的成交图片，然后借机推广公司的其他同事。</p>
    </div>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px;">
      <p style="margin:0;"><strong>2. 制造期待</strong>：巧妙推广"出差"机会，激发好奇心与向往</p>
      <p style="margin:8px 0 0 0; font-size:13px; font-style:italic;">例如：公司人少的时候，可以同时推广出差，产生对出差的盼望、好奇等。</p>
    </div>
  </div>
</div>

## ⏰ 二、核心环节：做好三个"半小时"

<div style="display:flex; flex-wrap:wrap; gap:12px; margin-bottom:16px;">
  <div style="background:rgba(102,126,234,0.08); padding:20px; border-radius:12px; flex:1; min-width:250px; box-shadow: 0 4px 15px rgba(102,126,234,0.15); border-top:4px solid #667eea;">
    <div style="font-size:36px; margin-bottom:8px;">👁️</div>
    <h4 style="margin:0 0 8px 0; color:#667eea;">阶段一：你做他看</h4>
    <p style="margin:0; font-size:13px;"><strong>时长</strong>：约30分钟<br><strong>目标</strong>：树立榜样，建立信心<br><strong>方法</strong>：组长亲自示范，新人近距离观摩学习<br><strong>关键点</strong>：组长的销售流程（五步法）必须简单、易学、有效</p>
    <div style="background:rgba(255,255,255,0.15); padding:8px; border-radius:8px; margin-top:8px;">
      <p style="margin:0; font-size:12px; font-style:italic;">操作要点：提醒新人不要离你太远，不然听不清楚，看不仔细。</p>
    </div>
  </div>
  <div style="background:rgba(118,75,162,0.08); padding:20px; border-radius:12px; flex:1; min-width:250px; box-shadow: 0 4px 15px rgba(118,75,162,0.15); border-top:4px solid #764ba2;">
    <div style="font-size:36px; margin-bottom:8px;">👀</div>
    <h4 style="margin:0 0 8px 0; color:#764ba2;">阶段二：你看他做</h4>
    <p style="margin:0; font-size:13px;"><strong>时长</strong>：约30分钟<br><strong>目标</strong>：检验学习成果，发现问题<br><strong>方法</strong>：让新人尝试，组长在旁观察</p>
    <div style="background:rgba(255,255,255,0.15); padding:8px; border-radius:8px; margin-top:8px;">
      <p style="margin:0; font-size:12px;"><strong>观察重点</strong>：<br>声音小 → 自信心不足<br>面无表情 → 缺少热情<br>表述不清 → 缺少思路</p>
    </div>
  </div>
  <div style="background:rgba(102,126,234,0.08); padding:20px; border-radius:12px; flex:1; min-width:250px; box-shadow: 0 4px 15px rgba(102,126,234,0.15); border-top:4px solid #667eea;">
    <div style="font-size:36px; margin-bottom:8px;">🤝</div>
    <h4 style="margin:0 0 8px 0; color:#667eea;">阶段三：边做边看</h4>
    <p style="margin:0; font-size:13px;"><strong>时长</strong>：持续跟进<br><strong>目标</strong>：即时辅导，建立信任</p>
    <div style="background:rgba(255,255,255,0.15); padding:8px; border-radius:8px; margin-top:8px;">
      <p style="margin:0; font-size:12px;"><strong>1. 主动介入</strong>：随时留意新人有没有遇到意向客户，不管能否成交都要表现出你在帮助他<br><strong>2. 事后复盘</strong>：离开客户后先鼓励、激励，再说明为什么没成交</p>
    </div>
  </div>
</div>

## 💪 三、心态建设：如何面对拒绝

<div style="display:flex; flex-wrap:wrap; gap:12px; margin-bottom:16px;">
  <div style="background:rgba(102,126,234,0.08); padding:16px; border-radius:12px; flex:1; min-width:280px; box-shadow: 0 4px 15px rgba(102,126,234,0.15);">
    <h4 style="color:#667eea; margin-top:0;">🧑‍💼 客户角度</h4>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px; margin-bottom:8px;">
      <p style="margin:0;"><strong>习惯性拒绝</strong>：客户对你不了解，拒绝是本能反应</p>
      <p style="margin:8px 0 0 0; font-size:13px; font-style:italic;">比喻：给小孩一张百元大钞他可能会当成一张纸扔掉，但是如果给他一颗糖，他会很高兴。</p>
    </div>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px; margin-bottom:8px;">
      <p style="margin:0;"><strong>价值未显</strong>：客户尚未认识到产品价值</p>
      <p style="margin:8px 0 0 0; font-size:13px; font-style:italic;">例子：我们去买衣服、鞋子之类的日用品都需要先试穿一下。</p>
    </div>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px;">
      <p style="margin:0;"><strong>应对</strong>：聚焦于一个简单、吸引人的核心卖点</p>
    </div>
  </div>
  <div style="background:rgba(118,75,162,0.08); padding:16px; border-radius:12px; flex:1; min-width:280px; box-shadow: 0 4px 15px rgba(118,75,162,0.15);">
    <h4 style="color:#764ba2; margin-top:0;">🪞 自身角度</h4>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px; margin-bottom:8px;">
      <p style="margin:0;"><strong>形象与热情</strong>：注意专业形象，展现热忱</p>
      <p style="margin:8px 0 0 0; font-size:13px; font-style:italic;">案例：肯德基老爷爷当初为了推销自己的炸鸡时，穿着一身雪白的西装。</p>
    </div>
    <div style="background:rgba(255,255,255,0.15); padding:12px; border-radius:8px;">
      <p style="margin:0;"><strong>技能熟练度</strong>：通过反复练习提升销售技巧</p>
    </div>
  </div>
</div>
