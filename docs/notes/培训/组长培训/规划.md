# 职业规划

## 一、 关于规划
*   **学业规划**：多数人只规划到大学，很少规划到研究生。这是因为上大学不是我们自己的想法和规划，而是身边人的絮叨，且大学期间的经历充满不确定性（“上大学就磕磕绊绊”）。
*   **缺乏规划的原因**：
    *   早期规划多由父母安排，个人缺乏自主性。
    *   对自身目标不清晰。例如，面试时可以说出过去的成绩，但当被问及未来规划时，却回答“不知道”，只是想找份工作安稳度日。
    *   人生变数太多，长期规划困难（“几十年中可能产生很多意外”）。
*   **职业起点**：许多人的第一份工作选择是“误打误撞”的，大多核心诉求是“离家近”、“感觉不错”，并没有长远的职业规划。

## 二、 关于价值与能力
*   **价值的本质**：社会运行的本质是价值交换。个人价值取决于做事产生的成果。
*   **能力界定**：能力是能够稳定地产出良好成果的本事。例如，能独立、完整地解决客户问题，就是一种能力。
*   **核心公式**：**想成为什么样的人 = 需要具备对应的能力 + 通过工作实践训练出来**。
*   **关系与实力**：依赖“关系”获得的工作可能不稳固（“关系已经不管用了”）。长久来看，最终依靠的是个人的实力和能力。

## 三、 关于工作与成长
*   **工作的意义**：在获得财务自由前，工作是不可或缺的。对年轻人而言，工作的首要目的不应该是当下赚多少钱（“只能赚辛苦钱，叫生存”），而是**获取和锻炼能力**，为未来（三年、五年后）创造价值。
*   **发展的定义**：发展不等于工龄增长或团队人数增加。发展的关键是**个人有进步、有成长、有能力上的改变和提升**。
*   **如何获得发展**：把当前的事情做好，做出成果。例如，作为组长，把人带好、培训好，自然能获得晋升机会（“今天把事情做好了，你不用管你当下没有得到，你肯定会（被）记得”）。
*   **年轻是资本**：20岁出头的年纪，精力充沛、学习能力强，是训练能力、接受挑战的黄金时期，不应计较短期得失或与旁人比较“轻松与否”。

## 四、 关于心态与成熟
*   **勇于尝试，不怕犯错**：能力的获取途径只有训练，要不怕做错，在挫折中总结经验（“跌倒又不可怕”）。
*   **接受不完美**：成熟就是知道事情不完美，但不影响你接受它。这包括接受不完美的父母、同事和公司，并给予尊重。
*   **区分事实与观点**：要能自己判断是非，对于已成事实的事情，无需过分在意他人的观点和建议。
*   **保持城府，专注自身**：不要私下诋毁或随意评价他人，专注做好自己的事，在过程中获得成长。即使现阶段收益有限，有意义的成长经历也胜过无意义的“贫穷”。
