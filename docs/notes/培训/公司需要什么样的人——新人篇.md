# 公司需要什么样的人—新人篇

公司招聘新人，不设严苛学历门槛，不苛求经验与能力，唯一且最重要的标准是——**态度**。  
经理绝不会让一个毫无态度的人进入团队，成为团队的“隐形阻碍”。  

## 什么是好态度？  

好态度无关表面的殷勤，不是刻意表现“我很喜欢这份工作”，更不需要用送礼讨好刷存在感。它藏在对选择的珍视里，对规则的敬畏中，对成长的渴望上。  


### 一、尊重公司：从认同到践行  
选择一家公司、一份职业，本质是选择一种生活方式。好态度的起点，是把这份选择当回事。  

- **守规则**：公司基础制度会在新人入群时明确告知。无规矩不成方圆，遵守制度不是束缚，是对团队秩序的基本维护。  
- **懂接纳**：入职前半天的观察期，是让你真实了解工作内容与形式的“试金石”。既然选择留下，就请放下预设的抱怨——既来之，则安之。  
- **跟流程**：集团所有分公司统一运作流程，没有“特例”。先执行再理解，在实践中消化，是融入团队的必经之路。  
- **重伙伴**：集团倡导“成就文化”，前辈对新人的包容与帮助是善意，但别把这份关怀当理所当然。在带教师傅面前摆架子、恃宠而骄，是态度的减分项。  


### 二、主动成长：在学习与突破中蜕变  
态度的另一面，是“不愿停在舒适区”的自觉。  

- **学专业**：产品知识、后台操作、业务流程、营销系统……岗位所需的核心技能，都需要主动钻研。  
- **改习惯**：从不敢提问到主动请教，从害怕与人接触到处变不惊，从松垮随意到注重形象——改变或许会带来短暂不适，但不改变，终将被问题反复折磨。  


**所谓好态度，不过是：选定了就认真对待，开始了就全力成长。**  
我们要的不是完美的人，而是愿意与公司共同进步的人。
