# 团队建设核心：建队先建己


## 一、引言：建队的渴望与现实困境
很多人都想拥有自己的团队，并且希望团队越大越好，人越多越好，实力越强越好。这是很自然的想法，因为团队壮大意味着更多可能：可以达标、晋升，未来甚至能开疆拓土。

**例如**：我们的兄弟公司正义合与羿凡，在西安经过一段时间的沉淀后，今年各自派出一位经理，合作在河南郑州开拓了一家全新的分公司。这就是通过建队实现发展的生动体现。

然而，一个普遍存在的症结是：**很多人，尤其是新晋或资历尚浅的组长，往往只知道“想建队”，却不清楚“为何建队”以及“建队的本质是什么”。**

## 二、核心论点：建队就是建自己
> **打铁还需自身硬，梧桐自有凤凰栖**

你的能力水平，决定了你能吸引和承载什么样的人才。团队建设的过程，本质上就是提升和完善自我的过程。

### 1. 业务能力是根基
必须具备两项核心业务能力：

*   **做好销售的能力**：不要求你时时顶尖，但**必须具备稳定、合格的销售水准**。关键是在团队需要时（例如达标冲刺期），你能站出来顶上。这体现在你能在需要的时候创造业绩。
*   **教别人做好销售的能力**：许多优秀的管理者自身销售并非最拔尖，但**擅长教授和赋能他人**。关键在于“会教”。
    *   **实例**：正义合的“焉总”，他个人销售未必最突出，但他能教会整个公司做好销售。更重要的是，在关键时期（例如带队出差冲刺达标时），他本人也能展现出强大的销售能力，证明自己“有做好销售的能力”。

**反思**：如果你的团队缺乏有战斗力的人，人数再多也如同“伪军”，看似人多却没有实际战斗力。历史证明，一小支有战斗力的精锐，可以击溃数倍于己的乌合之众。**兵不在多而在精，一将难求**。你首先需要问自己：**我能成为那个“将”吗？**

### 2. 欲为“师傅”，先成“将才”
在成为师傅之前，你首先是师傅团队中的“徒弟”。如果你希望自己的徒弟优秀、勇猛，那么你首先必须成为你师傅团队中那个能冲锋在前、勇于担责的“将才”式徒弟。

> **邓总个人经历**：以前在团队中，我始终要求自己和所带的“齐心队”要在公司保持领先水准，立志成为经理麾下的“将才”。正因为如此，我被赋予了更多责任和锻炼机会，例如曾连续带队出差六个星期，能力得到了飞速成长。

**扪心自问**：如果你自己都不是一个优秀的、能打硬仗的徒弟，凭什么要求你的徒弟优秀？

### 3. 榜样与习惯的力量
*   **榜样**：如果你自己都做不好销售，无法在徒弟面前成功“开单”，徒弟如何建立信心？你留不住真想做事的人，最终团队里只会剩下想混日子的人。
*   **习惯**：教销售，更多是**教态度、教习惯**。具体的方法和技巧，一个有心、想学好的人自己会去钻研。如果你自身习惯不好，当了“甩手掌柜”，那么团队里的人才都不是你培养的，团队自然也难以壮大变强。
*   **严重后果**：现实中不乏这样的案例：有人运气好，短期内团队迅速扩张到十几人，但因为没有战斗力，十几个人的业绩甚至比不上别人的两个人。与其耗费精力管理一群没有产出的人，不如先闭关修炼，强大自身。

### 4. 培训与学习能力
*   **会做培训**：要能系统地进行公司基础培训（如五步八点、企业文化、六大会议等），并能讲出其中的“道道”，为新人解决问题、赋予意义，而不是简单地说“习惯就好”。
*   **会听培训（学习能力）**：这是会做培训的前提。什么叫“会学习”？
    *   不止于机械地回复“收到”，而要能思考、举一反三、内化并表达感想。
    *   主动从各种有价值的内容中（如请教、阅读、励志文案）汲取营养，而非沉迷于无意义的消遣。
    *   **你的状态就是徒弟的镜子**：你开会不记笔记、进出随意，你的徒弟也会有样学样。你不能用自己所谓的“熟练”作为徒弟“可以不做好”的理由。当你还觉得做销售很难时，你其实还在“门外”看热闹，无法理解“门内”教人、带队的航道。

## 三、结论与行动指南
1.  **自身优秀是吸引力的源泉**：当你足够强大（销售稳定、会教、会学、习惯好），资源和人自然会向你靠拢。如果你每天都能有成交或接近成交，你还怕没有机会带新人、没有团队吗？
2.  **夯实基础，增强“消化”能力**：个人基本功不扎实，就像消化系统弱，给再多“食物”（新人）也吸收不了，反而成为负担。**先专注修炼，强大自身**。当你的“消化系统”足够强，你才能有效地培养更多人。
3.  **公平与机会**：公司的资源（如新观察日）会优先倾向于有成果（如当日有成交）的组长。这是对拼搏者的公平。没有成交，意味着你可能暂时只具备“帮带”的资格，而非拥有。
4.  **最终的叩问**：请时刻反思：如果今天新人没有跟你一起作业，那个成交的顾客，**你自己能搞定吗**？如果答案是否定的，说明你的个人能力仍需锤炼。

**行动号召**：从今天起，请聚焦自身成长，用更高的标准要求自己。建队之路，始于强大的自己。当你成为一棵挺拔的梧桐树，凤凰自然会来栖息。兄弟们，加油！
