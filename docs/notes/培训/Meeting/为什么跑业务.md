## 二、问题的本质：认知与现实的差距 🧩

<div style="background:rgba(102,126,234,0.06);border:1px solid rgba(102,126,234,0.15);padding:20px;border-radius:12px;margin:20px 0;">
<div style="background:rgba(102,126,234,0.08);padding:15px;border-radius:8px;border-left:4px solid #667eea;">
<p class="tc-body" style="font-size:15px;margin:0;line-height:1.7">💡 <strong class="tc-blue">问题的定义</strong>：问题 = 现状与期望之间的差距。解决问题的过程，就是<strong class="tc-blue">缩小这一差距</strong>的过程。</p>
</div>
</div>

### 问题存在的两种形式

<div style="display:flex;flex-wrap:wrap;gap:12px;margin:20px 0;">

<div class="tc-body" style="background:rgba(239,68,68,0.06);border:1px solid rgba(239,68,68,0.15);;padding:20px;border-radius:12px;flex:1;min-width:280px">
<p class="tc-red tc-bold" style="font-size:20px;;margin:0 0 12px 0">⚠️ 障碍型问题</p>
<p style="font-size:14px;margin:0 0 8px 0;line-height:1.6;">现状 < 期望</p>
<p class="tc-sm" style="margin:0;line-height:1.5">例如：业绩不达标、客户投诉、团队士气低落</p>
</div>

<div class="tc-body" style="background:rgba(34,197,94,0.06);border:1px solid rgba(34,197,94,0.15);;padding:20px;border-radius:12px;flex:1;min-width:280px">
<p class="tc-green tc-bold" style="font-size:20px;;margin:0 0 12px 0">🚀 机会型问题</p>
<p style="font-size:14px;margin:0 0 8px 0;line-height:1.6;">现状 < 潜在期望</p>
<p class="tc-sm" style="margin:0;line-height:1.5">例如：如何提升效率、如何开拓新市场、如何培养更多骨干</p>
