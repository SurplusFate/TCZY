## 二、常用功能说明

![联通公众功能](./images/1000004085.jpg)

| 功能 | 说明 |
|------|------|
| **笔算智选** | 查询陕西联通号码的基本情况：近三个月实际消费、主副卡、有无宽带等 |
| **特殊用户激活** | 查询该用户是否为陕西黑名单用户 |
| **360视图（红牌）** | 查询该用户是否为红黄牌用户 |
| **商品受理-移网** | 受理单卡套餐（无宽带） |
| **商品受理-融合** | 受理融合套餐（有宽带） |
| **加装副卡** | 新开副卡或携转为副卡 |
| **总部宽带资源预判** | 查询宽带资源 |
| **携号转网状态查询** | 查询转网状态，转入或转出均能查询 |
| **快速缴费** | 充话费，只能给本地号码充值 |

## 三、信息认证常见问题

### 3.1 身份证无法读取

**解决方案**：尝试使用家人身份证读取。

### 3.2 黑名单

#### 黑名单查询

![黑名单查询](./images/1000251721.jpg)

可利用 **商品受理-融合-施工中开户** 查询到黑名单具体号码。

![黑名单查询2](./images/1000251722.jpg)

再利用手机拨号功能（苹果手机不行）或者浏览器搜索可查询到具体归属地。

![归属地查询](./images/1000259945.png)

![归属地查询2](./images/1000259943.png)

#### 黑名单处理方式

| 类型 | 处理方式 |
|------|----------|
| **本地黑名单** | 金额小于 **100元** 或时间大于 **5年** 的，可直接解除 |
| **外地黑名单** | 如果对应归属地有分公司，可联系TC售后客服帮忙解除（如重庆、杭州等） |
| **黑名单通解** | 打 **10010** 客服申诉；部分黑名单可利用联通APP销户及停机号码交费 **0.01元** 消除 |
| **特殊黑名单** | **信安黑名单**（涉诈用户），需找公安局开具证明，**基本无法办理** |

![黑名单处理](./images/1000004102.jpg)

### 3.3 红黄牌

#### 红牌产生原因

| 原因 | 说明 |
|------|------|
| **短时间内频繁开户销户** | **30天内**累计销户 **≥2次** |

![红牌-频繁销户](./images/1000004106.jpg)

| 原因 | 说明 |
|------|------|
| **名下存在异常号码** | 名下已有 **1个及以上**号码（不区分主副卡）处于局方停机、局方半停机状态 |

![红牌-异常号码](./images/1000004105.jpg)

#### 黄牌产生原因

| 原因 | 说明 |
|------|------|
| **名下存在异常号码** | 名下已有 **1个**号码处于局方停机、局方办停机状态 |

![黄牌](./images/1000004109.jpg)

| 原因 | 说明 |
|------|------|
| **低高龄用户** | 用户年龄过低或过高 |

![低高龄用户](./images/1000004111.jpg)

#### 红黄牌影响

| 牌型 | 是否影响新入网 | 是否影响携转 |
|------|---------------|-------------|
| **红牌** | ❌ **无法办理新入网** | ✅ 不影响 |
| **黄牌** | ⚠️ 部分受限 | ✅ 不影响 |
| **低高龄用户** | ✅ **不影响入网** | ✅ 不影响 |

> ⚠️ **重要**：**红黄牌不影响携号转网**。

### 3.4 身份信息不一致

![身份不一致](./images/1000004113.jpg)

**原因**：该用户改过名

**解决方案**：需到营业厅更新实名信息。

### 3.5 一证多户

#### 一证五户

![一证五户](./images/1000004117.jpg)

- **原因**：名下已有 **5个**联通号码
- **解决方案**：需要销户

#### 一证十户

![一证十户](./images/1000004116.jpg)

- **原因**：名下已有 **10个**号码（大部分是虚拟运营商号码）
- **解决方案**：需要销户
- **查询方法**：微信或支付宝搜索 **"一证通查"** 可查询名下号码数量及运营商（填写的姓名、身份证号、手机号都必须是本人的）

### 3.6 人脸识别不通过

![人脸识别](./images/1000004119.jpg)

**解决方案**：换背景、换角度多尝试。

## 四、携转过程常见问题

### 4.1 手机号输错

![手机号输错](./images/1000004123.jpg)

**解决方案**：核对号码后重新输入。

### 4.2 号码非本人

![号码非本人](./images/1000004125.jpg)

**解决方案**：确认号码实名信息是否为本人，如非本人需先过户。

### 4.3 携转码输错

![携转码输错](./images/1000004124.jpg)

> ⚠️ **特别注意**：携转码输错**系统不会第一时间反馈**，但在给客户重新做携转时会弹出**在途单**。

**携转过程出现的问题有时系统不会第一时间反馈，需要在走完携转流程后第一时间使用「携号转网状态查询」功能复查！**

![携转查询](./images/1000004133.jpg)

#### 携转状态判断

**正确状态** ✅

![携转正确](./images/1000004130.jpg)

**错误状态** ❌

![携转错误](./images/1000004128.jpg)

#### 携转失败处理

| 情况 | 处理方式 |
|------|----------|
| **可更正** | 让后台操作更正携转码或用户信息，**白卡即可生效使用，无需重新写卡** |
| **无法更正** | 需要撤单重新走流程，**重新写卡** |

## 五、录宽带过程常见问题

### 5.1 忘记备注

**后果**：宽带师傅会打电话要求撤单重录。

**建议**：录入时务必填写完整备注信息。

### 5.2 使用总部地图选址导致宽带账号申请失败

![宽带选址](./images/1000004135.jpg)

**解决方案**：

1. 先用 **总部地图选址** 找到具体的装机地址
2. 再切回 **标准地址** 下单

- 操作演示视频：[阿里云盘链接](https://www.alipan.com/s/fr7TBKRQQQZ)
- 点击链接保存，或复制本段内容，打开「阿里云盘」APP，无需下载极速在线查看

## 六、问题速查表

### 信息认证问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 身份证无法读取 | 身份证芯片或读卡器问题 | 尝试家人身份证 |
| 黑名单 | 欠费停机等 | 本地：金额<100或>5年可解；外地：联系TC售后 |
| 信安黑名单 | 涉诈用户 | 需公安局开具证明，基本无法办理 |
| 红牌 | 30天销户≥2次或异常号码 | 无法新入网，但不影响携转 |
| 黄牌 | 1个号码异常停机或低高龄 | 部分受限，但不影响携转 |
| 身份信息不一致 | 用户改过名 | 到营业厅更新实名信息 |
| 一证五户 | 名下5个联通号码 | 销户 |
| 一证十户 | 名下10个号码（多为虚商） | 销户，用"一证通查"查询 |
| 人脸识别不通过 | 光线、角度问题 | 换背景换角度多尝试 |

### 携转过程问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 手机号输错 | 输入错误 | 核对后重新输入 |
| 号码非本人 | 实名信息不符 | 先过户再携转 |
| 携转码输错 | 输入错误 | 系统不即时反馈，需用状态查询复查 |
| 携转失败 | 信息有误 | 可更正则后台修正（无需重新写卡）；否则撤单重来 |

### 录宽带问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 忘记备注 | 录入时遗漏 | 宽带师傅会要求撤单重录 |
| 宽带账号申请失败 | 使用总部地图选址 | 先用总部地图选址定位，再切回标准地址下单 |
