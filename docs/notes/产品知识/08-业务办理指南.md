## 一、新开号码业务

### 1.1 保号新开

#### 适用场景

用户原号码存在以下情况之一：
- 非西安号码
- 非本人实名
- 不愿意转网

#### 办理方案

```
┌─────────────────────────────────────────────────────────┐
│                    保号新开流程                          │
├─────────────────────────────────────────────────────────┤
│  ① 原号码降档 → ② 新办联通号码 → ③ 享受新套餐优惠        │
└─────────────────────────────────────────────────────────┘
```

| 运营商 | 最低套餐 |
|--------|----------|
| **中国联通** | 8元套餐 |
| **中国移动** | 8元套餐 |
| **中国电信** | 5元套餐 |

### 1.2 保宽带新开

#### 适用场景

用户宽带地址无联通资源

#### 办理方案

```
┌─────────────────────────────────────────────────────────┐
│                   保宽带新开流程                         │
├─────────────────────────────────────────────────────────┤
│  ① 保留原宽带 → ② 原套餐降最低档 → ③ 新办联通号码       │
└─────────────────────────────────────────────────────────┘
```

## 二、携号转网业务

### 2.1 业务类型

| 类型 | 适用场景 | 说明 |
|------|----------|------|
| **直接携转** | 用户原号码无任何合约、宽带等业务限制 | 直接办理携入 |
| **新开携转** | 用户原号码存在无法立即取消的业务 | 先保留原业务，再办理携入 |

### 2.2 核心注意事项

#### 资格查询与授权码获取

| 项目 | 说明 |
|------|------|
| **短信格式** | `SQXZ#姓名#身份证号码` |
| **发送至** | 电信 **10001**，移动 **10086**，联通 **10010** |
| **特殊说明** | 部分老用户需使用**一代身份证号码** |

#### 办理时机与生效规则

| 规则 | 说明 |
|------|------|
| **生效机制** | **整点生效**（如14:02办理，15:00生效） |
| **晚上8点后** | 次日早上 **8:00** 生效 |
| **月底最后一天20点后** | 必须选择"**立即生效**" |
| **生效判定** | 原手机卡信号消失 |

#### 生效后业务限制

| 限制项 | 说明 | 解决方案 |
|--------|------|----------|
| **锁定期** | 转入后 **120天** 内无法转出 | — |
| **短信功能** | 携入后 **2小时内** 无法接收联通验证码 | 使用数据流量"**一键登录**"联通APP |
| **主副卡** | 携入主卡时**不能同步办理副卡** | 携入成功后再办理 |

#### 费用与账户处理

##### 原话费退还

| 项目 | 说明 |
|------|------|
| **时间** | 次月 **6号** 后 |
| **地点** | 原运营商官方营业厅 |
| **期限** | 不超过 **90天** |

##### 缴费提醒

| 运营商 | 退费方式 |
|--------|----------|
| **中国移动** | 携出3天后可在移动APP操作携出退费，退至**另一个陕西移动号码** |
| **中国联通/电信** | 到官方营业厅办理退费 |

> 💡 **缴费建议**：优先使用 **联通公众APP / 中国联通APP**

#### 靓号特别规则

| 规则 | 说明 |
|------|------|
| **首次转入** | **不执行**靓号协议 |
| **过户** | 将**触发联通靓号规则**（需签靓号协议） |

#### 业务查询与核实

| 方式 | 路径 |
|------|------|
| **携转状态查询** | 联通公众APP → 移网 → 携号转网状态查询 |
| **成功验证** | 尝试缴费，能识别用户资料即成功 |

## 三、关键信息速查

### 3.1 短信指令速查

| 运营商 | 查询资格 | 获取授权码 |
|--------|----------|-----------|
| **中国移动** | `CXXZ#姓名#身份证号` → 10086 | `SQXZ#姓名#身份证号` → 10086 |
| **中国联通** | `CXXZ#姓名#身份证号` → 10010 | `SQXZ#姓名#身份证号` → 10010 |
| **中国电信** | `CXXZ#姓名#身份证号` → 10001 | `SQXZ#姓名#身份证号` → 10001 |

### 3.2 生效时间速查

| 办理时间 | 生效时间 |
|----------|----------|
| 工作日 8:00-20:00 | 下一个整点 |
| 工作日 20:00后 | 次日 8:00 |
| 月底最后一天 20:00后 | 必须选"立即生效" |

### 3.3 费用标准速查

| 项目 | 标准 |
|------|------|
| **联通/移动最低套餐** | 8元/月 |
| **电信最低套餐** | 5元/月 |
| **话费退还期限** | 次月6号后，90天内 |
| **携转锁定期** | 120天 |
| **短信恢复时间** | 携入后约2小时 |
