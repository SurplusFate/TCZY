## 一、什么是携号转网

**携号转网**是指电话号码的持有人不再使用原运营商的电信服务，而想加入另一个电信运营商时，可以**不改变原来的电话号码**，将这个电话号码转入另一个运营商的旗下。

简单来说，就是消费者可以在**不更换手机号码**的情况下，**更换自己的电信运营商**，享受不同运营商提供的各种服务。

![携号转网](./images/1000002917.jpg)

### 适用范围

| 支持转网 | 不支持转网 |
|----------|-----------|
| 中国移动、中国联通、中国电信的个人手机号码 | 虚拟运营商号码（170、171、162、165、167等） |
| 同一归属地之间互转 | 跨地市转网 |
| | 卫星移动业务号码（1349、174等） |
| | 物联网号码（146、148、149、140-144等） |

## 二、办理条件

根据工信部《携号转网服务管理规定》，用户办理携号转网需同时满足以下条件：

| 条件 | 说明 |
|------|------|
| ✅ **实名登记** | 号码已在原运营商办理真实身份信息登记 |
| ✅ **正常使用** | 号码处于正常使用状态（非挂失、停机等） |
| ✅ **结清费用** | 与原运营商结清所有已出账电信费用，如有未出账费用需约定缴费方式 |
| ✅ **无合约限制** | 与原运营商无在网约定期限限制的协议，或已解除在网期限限制 |
| ✅ **满120日** | 距离最近一次携号转网已满 **120日** |

### 查询是否符合条件

编辑短信发送至当前运营商客服热线：

```
CXXZ#用户名#证件号码
```

| 运营商 | 客服热线 | 短信发送号码 |
|--------|----------|-------------|
| 中国移动 | 10086 | 10086 |
| 中国联通 | 10010 | 10010 |
| 中国电信 | 10000 | 10001 |

### 3.3 办理所需材料

到目标运营商办理携入时，需提供：

2. **有效的携出授权码**
3. **申请携出号码对应的移动电话卡**

## 四、注意事项

### 4.1 转网前须知

| 事项 | 说明 |
|------|------|
| **话费余额** | 转网完成后，原运营商话费余额可按注销规则退还，具体咨询原运营商客服 |
| **积分/权益** | 原运营商的积分、会员权益等将清零，建议提前兑换使用 |
| **绑定业务** | 银行、支付宝、微信等绑定的手机号需重新验证 |
| **靓号协议** | 如号码属于靓号，转网后可能需要重新签订靓号协议 |

### 4.2 转网后影响

| 影响项 | 说明 |
|--------|------|
| **短信接收** | 转网生效后约 **1小时内**，可能无法正常接收短信（银行验证码等） |
| **APP登录** | 部分APP可能因运营商信息变更需要重新验证 |
| **通话功能** | 网络切换期间可能短时间影响正常通信 |
| **业务办理** | 转网后约 **1小时内** 无法登录运营商APP办理业务 |

### 4.3 靓号特殊规定

| 情形 | 规则 |
|------|------|
| **协议期内转网** | 需支付违约金（剩余合约期最低消费总额的90%左右） |
| **协议期满后转网** | 首次转出时可不执行靓号协议 |
| **转网后过户** | 若后续再过户，将重新触发靓号协议 |

## 五、常见问题

### Q1：携号转网收费吗？

**免费**。携号转网服务本身不收取任何费用。

### Q2：转网后还能转回去吗？

**可以**。但需要距离上次转网满 **120日** 后才能再次办理。

### Q3：转网后号码归属地会变吗？

**不会**。携号转网只能在**同一归属地**内进行，号码归属地保持不变。

### Q4：转网期间能正常使用手机吗？

- **生效前**：原移动电话卡正常使用，由原运营商提供服务
- **生效时**：网络切换期间可能短时间影响通信
- **生效后**：新电话卡激活启用，由新运营商提供服务

### Q5：遇到运营商阻挠怎么办？

可向以下渠道投诉：
- 运营商客服热线（电信10000/10001，移动10086，联通10010）
- 工信部申诉受理机构
- 互联网信息服务投诉平台
