## 一、手机终端（硬件与设计）

手机的硬件配置、天线设计及布局直接影响其接收信号的能力。

### 典型案例：苹果"死亡之握"

指当用户以特定方式握持手机时，会导致信号强度急剧下降。这主要暴露了手机天线设计在特定场景下的缺陷。

> 💡 **提示**：不同品牌、不同型号的手机，天线设计和信号接收能力存在差异。选购手机时可关注其信号评测。

## 二、运营商网络（核心因素）

运营商建设的**基站数量**、**覆盖密度**和**网络质量**是决定信号强度的根本。

### 2.1 历史背景：4G时代基站数量差异

在4G时代，三大运营商的基站建设规模存在显著差距，这直接影响了当时的用户体验和品牌印象。

| 运营商 | 4G基站数量（历史数据） |
|--------|----------------------|
| **中国移动** | 241万座 |
| **中国电信** | 138万座 |
| **中国联通** | 99万座 |

### 2.2 现状：共建共享成为新趋势

为了**节约成本**、**提高效率**、**加速网络覆盖**，运营商之间开始了大规模的"**共建共享**"合作。

#### 中国联通 & 中国电信

| 项目 | 详情 |
|------|------|
| **合作起始** | 2019年起，双方共同共建共享5G网络 |
| **5G共享基站** | 累计开通 **100万个**，实现全国所有城市、县城、乡镇及重点发达农村的覆盖 |
| **4G共享基站** | 开通 **110万站**，极大地弥补了联通和电信在4G覆盖上的短板 |

#### 中国移动 & 中国广电

| 项目 | 详情 |
|------|------|
| **合作策略** | 采用"**700MHz + 2.6GHz**"协同建网策略 |
| **合作内容** | 共同建设5G网络 |

## 三、环境与其它因素

### 3.1 建筑遮挡

| 场景 | 影响 |
|------|------|
| **钢筋水泥结构** | 对信号造成严重衰减 |
| **地下空间** | 地下室、地铁等场景信号覆盖困难 |
| **高层建筑** | 信号在室内分布不均，高层可能出现"信号孤岛" |

### 3.2 距离基站

距离基站越远，信号越弱。在偏远地区或基站覆盖边缘，信号质量会明显下降。

### 3.3 网络拥堵

在人口密集区域（如商场、体育场、演唱会现场），同时上网的用户过多，会导致：
- 网速下降
- 通话质量变差
- 甚至无法接入网络

### 3.4 基站干扰与居民反对

| 问题 | 说明 |
|------|------|
| **基站间信号干扰** | 基站布局不合理或频率规划不当，会导致信号干扰 |
| **居民抵制** | 部分居民对基站的电磁辐射存在顾虑，抵制基站建设，导致局部信号覆盖不佳 |

## 四、总结与建议

### 4.1 信号问题排查思路

```
信号不好？
    │
    ├─ 换手机测试 → 手机问题 → 考虑更换终端
    │
    ├─ 换地点测试 → 环境问题 → 移动到开阔地带或窗边
    │
    └─ 询问周围用户 → 运营商问题 → 考虑携号转网或投诉
```

### 4.2 改善信号的建议

| 场景 | 建议措施 |
|------|----------|
| **室内信号差** | 靠近窗户、使用WiFi通话、安装信号放大器 |
| **地下室/电梯** | 提前下载离线内容、使用WiFi网络 |
| **偏远地区** | 选择覆盖更好的运营商、使用卫星电话作为备用 |
| **人口密集区** | 避开高峰时段、使用5G网络（相对4G容量更大） |

### 4.3 运营商选择参考

| 需求场景 | 推荐考虑 |
|----------|----------|
| **追求覆盖广** | 中国移动（基站数量最多） |
| **追求性价比** | 中国联通/电信（共建共享后覆盖大幅提升） |
| **5G体验** | 移动/广电（700MHz频段覆盖能力强） |

## 附录：信号问题速查

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 信号满格但网速慢 | 网络拥堵 | 切换5G/4G、避开高峰时段 |
| 信号时有时无 | 基站切换/建筑遮挡 | 移动位置、重启手机 |
| 只有2G信号 | 偏远地区/基站故障 | 向运营商反馈、考虑携号转网 |
| 室内信号差、室外正常 | 建筑遮挡 | 使用WiFi通话、信号放大器 |
