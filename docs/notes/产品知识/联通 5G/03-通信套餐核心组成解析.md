## 四、网速知识详解

### 4.1 核心概念：网速单位换算

网络速率的基本单位是 <strong>比特/秒 (bit/s, bps)</strong>，而文件大小的单位是 <strong>字节 (Byte)</strong>。

> <strong>核心关系：1 Byte = 8 bits</strong>

<div style="display:flex;flex-wrap:wrap;gap:12px;margin:20px 0">

<div class="tc-card tc-card-blue" style="flex:1;min-width:280px;padding:15px">
<p class="tc-muted" style="margin:0 0 8px 0;font-size:14px">📊 换算关系</p>
<p class="tc-blue tc-bold" style="margin:0;font-size:16px">1 MB/s = 8 Mbps</p>
<p class="tc-muted tc-sm" style="margin:5px 0 0 0">下载速度与带宽速率的换算。</p>
</div>

<div class="tc-card tc-card-green" style="flex:1;min-width:280px;padding:15px">
<p class="tc-muted" style="margin:0 0 8px 0;font-size:14px">📊 换算关系</p>
<p class="tc-green tc-bold" style="margin:0;font-size:16px">1 GB/s = 8 Gbps</p>
<p class="tc-muted tc-sm" style="margin:5px 0 0 0">同上，更大单位。</p>
</div>

</div>

- <strong>行业惯例</strong>：运营商宣传的宽带速率（如1000M）单位是 <strong>Mbps</strong> 或 <strong>Gbps</strong>，通常按 <strong>1000进制</strong>计算；设备显示的下载速度单位是 <strong>MB/s</strong>。
- <strong>简易公式</strong>：`理论最高下载速度 (MB/s) ≈ 宽带套餐速率 (Mbps) ÷ 8`。

> 💡 <strong>例如</strong>：1000M宽带 ≈ 1000 ÷ 8 = <strong>125 MB/s</strong> 的下载速度。
