## 四、网速知识详解

### 4.1 核心概念：网速单位换算

网络速率的基本单位是 <strong>比特/秒 (bit/s, bps)</strong>，而文件大小的单位是 <strong>字节 (Byte)</strong>。

> <strong>核心关系：1 Byte = 8 bits</strong>

<div style="display:flex;flex-wrap:wrap;gap:12px;margin:20px 0;">

<div style="flex:1;min-width:280px;background:rgba(102,126,234,0.06);border:1px solid rgba(102,126,234,0.15);padding:15px;border-radius:12px;">
<p style="margin:0 0 8px 0;font-size:14px;color:#666;">📊 换算关系</p>
<p style="margin:0;font-size:16px;font-weight:bold;color:#667eea;">1 MB/s = 8 Mbps</p>
<p style="margin:5px 0 0 0;font-size:13px;color:#666;">下载速度与带宽速率的换算。</p>
</div>

<div style="flex:1;min-width:280px;background:rgba(34,197,94,0.06);border:1px solid rgba(34,197,94,0.15);padding:15px;border-radius:12px;">
<p style="margin:0 0 8px 0;font-size:14px;color:#666;">📊 换算关系</p>
<p style="margin:0;font-size:16px;font-weight:bold;color:#22c55e;">1 GB/s = 8 Gbps</p>
<p style="margin:5px 0 0 0;font-size:13px;color:#666;">同上，更大单位。</p>
</div>

</div>

- <strong>行业惯例</strong>：运营商宣传的宽带速率（如1000M）单位是 <strong>Mbps</strong> 或 <strong>Gbps</strong>，通常按 <strong>1000进制</strong>计算；设备显示的下载速度单位是 <strong>MB/s</strong>。
- <strong>简易公式</strong>：`理论最高下载速度 (MB/s) ≈ 宽带套餐速率 (Mbps) ÷ 8`。

> 💡 <strong>例如</strong>：1000M宽带 ≈ 1000 ÷ 8 = <strong>125 MB/s</strong> 的下载速度。
