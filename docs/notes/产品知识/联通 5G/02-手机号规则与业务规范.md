## 一、运营商分类

中国的电信运营商主要分为以下两类：

### 1.1 四大国有运营商（基础电信企业）

| 运营商 | 主要业务特点 |
|--------|-------------|
| **中国移动** | 移动通信领域用户规模最大 |
| **中国联通** | 全业务运营，固网移动并重 |
| **中国电信** | 固网和宽带领域传统强者 |
| **中国广电** | 2022年入局，5G新竞争者 |

### 1.2 全国虚拟运营商

- **数量**：众多，如**阿里通信**、**小米移动**、**京东通信**等
- **特点**：租用基础运营商网络，提供差异化服务
- **参考名单**：[知乎专栏 - 虚拟运营商名单](https://zhuanlan.zhihu.com/p/413595342)

## 三、重要说明与历史问题

### 3.1 虚拟运营商（虚商）说明

| 时期 | 管理状态 | 现状 |
|------|----------|------|
| **2024年以前** | 管理相对宽松，实名制要求不严 | 部分用户名下被办理大量虚商卡 |
| **2024年以后** | 监管极大加强，严格落实实名制 | 虚商号码**不占用**"10张"基础运营商名额 |

### 3.2 其他政策细节

| 政策项 | 规则说明 |
|--------|----------|
| **注销冷却期** | 手机号码注销后，需间隔 **30天** 才能重新办理新卡 |
| **年龄限制** | 未满 **18周岁** 的用户**无法独立**开办手机卡 |

## 四、自查名下号码方法

使用官方 **"一证通查"** 服务进行免费查询：

### 查询途径

| 平台 | 操作方式 |
|------|----------|
| **微信** | 搜索"**工信微报**"或"**一证通查**"官方公众号/小程序 |
| **支付宝** | 首页搜索框搜索"**一证通查**"服务 |

### 覆盖范围

查询结果将列出您名下在**所有基础运营商**和**虚拟运营商**处注册的手机号码。

> ⚠️ **提示**：若发现非本人使用或不需要的号码，请**尽快联系相应运营商进行注销**，以免影响办理新业务或产生潜在风险。

```
┌─────────┬─────────┬─────────┐

│ 号段(运营商)│ 归属地编码 │ 个人编码  │
└─────────┴─────────┴─────────┘
```

## 六、靓号业务规范

> 注：以下规范以**西安地区**为例，其他地区可能存在差异

### 6.1 靓号类型

靓号（规范名称：**特号**）主要包括：

| 类型 | 说明 | 示例 |
|------|------|------|
| **生日号** | 与生日日期匹配 | 如19900815 |
| **连续号** | 数字连续递增或递减 | 如123456、654321 |
| **豹子号** | 后几位数字相同 | 如666、8888 |
| **四连号** | 后四位数字相同 | 如6666、8888 |
| **重复号** | 单重复号、双重复号等规律号码 | 如1212、6688 |
| **区号号** | 包含城市区号 | 如029XXXX |

### 6.2 推广策略与风险提示

| 策略 | 说明 |
|------|------|
| **不作为主推** | 仅在节假日等特定时点不定期投放，资源有限 |
| **资源不确定性** | 号码资源紧缺，难以保证完全投合用户喜好 |

### 6.3 重要协议条款

靓号业务附有**严格协议**，务必向用户清晰说明：

#### 核心约束

| 约束项 | 要求 |
|--------|------|
| **每月最低消费** | 用户需承诺每月达到指定最低消费额度 |
| **长期在网** | 通常要求 **10年** 以上 |

#### 违约后果

| 违约情形 | 违约金计算 |
|----------|-----------|
| **提前解约**（销户或协议期内携号转网） | 剩余合约期累计最低消费总额的 **90%**（具体以签署协议为准，可能存在30%等其他比例） |

> ⚠️ **特别注意**：此操作后期处理隐患大，务必谨慎！

#### 虚假靓号协议识别

针对电信用户遇到的虚假靓号协议问题，以下两种情况需特别留意：

1. **无明显规律的号码**：部分电信号码本身并无特殊规律，但合约中却包含靓号协议
2. **2019年之前入网的号码**：即使号码看起来符合靓号特征，也可能存在类似情况

**复查方法**：通过电信官方APP的"**一键诊断**"功能进行复查。若诊断结果中未显示相关协议，则可认定该靓号协议为**虚假协议**，不具备实际效力。

### 6.4 靓号过户及转网规则

| 情形 | 规则 |
|------|------|
| **协议期内/期满后过户** | 新持有者需按当前最新靓号协议，重新履行最低消费和协议期限 |
| **协议期后携号转网** | 首次转出时可不执行靓号协议；若后续再过户，将重新触发靓号协议 |

### 6.5 增值业务收费标准

| 费用类型 | 是否计入低消 | 说明 |
|----------|-------------|------|
| **套餐费用** | ✅ 计入 | 手机卡套餐部分 |
| **套餐外通话/流量** | ✅ 计入 | 超出套餐的通话和流量费用 |
| **摄像头7天回看** | ❌ 不计入 | 增值业务，单独计费 |
| **宽带提速** | ❌ 不计入 | 增值业务，单独计费 |
| **云盘会员** | ❌ 不计入 | 增值业务，单独计费 |

### 6.6 操作流程提醒

1. **选号时**：用户可通过输入完整号码搜索，系统会明确展示该靓号的**月最低消费**和**年限要求**
2. **签约前**：务必引导用户**预览并讲清《靓号协议》**的所有规则，确保用户完全理解并同意
3. **过户时**：协议期内的靓号办理过户，新用户通常需要**重新签订**靓号协议并遵守新的消费标准
